表 5-23 “神经元网络模型ChatGPT/ChatGLM/Qwen”SFT QLoRA微调成为“教育AI模型ChatEdu”的超参数设置案例

• “output_dir”: 模型训练结果的保存目录。

• “model_name_or_path”: 使用的预训练模型的名称或路径。

• “train_file”: 训练数据文件的路径。

• “num_train_epochs”: 训练的轮数（全部数据遍历一次为1个epoch）。Epoch 数据量比较少，才200-300条回答数据，为了训练效果，设置得比较多，但如果数据量非常大，达到万级、十万的数据量，即使拥有非常强的计算硬件，也要考虑将epoch控制在两位数甚至一位数。

• “per_device_train_batch_size”: 每个设备（显卡）上的训练批处理大小。

• “gradient_accumulation_steps”: 梯度累积的步数。

• “learning_rate”: 学习率。学习率通常不会直接影响显存占用。学习率是深度学习模型训练中的一个超参数，用于控制每次参数更新的步长。它决定了模型在训练过程中对于梯度信息的利用程度。一般来说，较大的学习率会使模型更快地收敛，但可能会导致训练过程不稳定。较小的学习率则可以提高模型的训练稳定性，但可能会增加训练时间。

• “max_seq_length”: 输入序列的最大长度。

• “logging_steps”: 训练过程中记录日志的步数。

• “save_steps”: 模型保存的步数。

• “save_total_limit”: 保存模型的最大数量。

• “lr_scheduler_type”: 学习率调度器的类型。

• “warmup_steps”: 学习率预热的步数。

• “QLoRA_rank”: QLoRA（Locally Recurrent Autoencoder）的相关参数之一。

• “QLoRA_alpha”: QLoRA 的相关参数之一。

• “QLoRA_dropout”: QLoRA 的相关参数之一。

• “gradient_checkpointing”: 是否使用梯度检查点技术。

• “disable_tqdm”: 是否禁用 tqdm 进度条显示。

• “optim”: 损失调优函数的类型。

• “seed”: 随机种子。

• “fp16”: 是否使用半精度浮点数运算进行训练。

• “report_to”: 结果报告输出的目标。

• “dataloader_num_workers”: 数据加载器使用的并发工作进程数。

• “save_strategy”: 模型保存策略。

• “weight_decay”: 权重衰减。

• “max_grad_norm”: 梯度裁剪的最大范数。

• “remove_unused_columns”: 是否移除不使用的列。

namespace SemanticKernel_AI_LLM_Try.Controllers

{

    public class AskGPTTranslationController : Controller

    {

        [HttpGet(Name = "AskGPTTranslation")]

        public async Task<string> AskGPTTranslation(String inputString)

        {

            IKernel iKernel = Kernel.Builder.Build();//新生一个IKernel对象类型的iKernel对象实例

            ISKFunction llm = iKernel.CreateSemanticFunction("请将'"+inputString+"'翻译成为英文。"); //创建一个SemanticFunction实现模型的调用。SemanticFunction函数的输入是AI模型宿主的Prompt语言（目前Prompt语言最好是遵守ICIO语法框架）。在此的Prompt语言调用AI模型进行中英翻译。当然也可进一步调用其他各种功能。

            SKContext? responseFromLLM = await llm.InvokeAsync(inputString);

            return responseFromLLM.Result;

        }

    }

一个复杂混沌系统整体，由众多简单有序系统组成，其中，简单有序系统的数量越多、网络结构层次深度越多，复杂混沌系统整体就越能涌现大概率规律。涌现的大概率规律成功解决了需求问题则可称为“顿悟”。另外，混沌也常被看作是系统认知过程中的早期整体朦胧状态。

每个简单有序系统可由数学函数准确预测规律，但是，所有的简单有序系统的数学函数的平均权重的准确预测，可能不是复杂混沌系统的真实逻辑规律，所有的简单有序系统的数学函数的自主接受训练学习修正权重的大概率预测，才是复杂混沌系统的大概率逻辑规律。例如，蝴蝶效应；又如，人脑神经网络复杂系统；再例如，神经网络人工智能，一个神经元并没有所谓的“智慧”，但超过1000亿级的神经元整合在一起之后，可以涌现我们人类的高级智慧。计算机隐喻人脑的超过1000亿级的人工神经元的神经网络，则可涌现人工智能，从而辅助人解决需求问题。即，通过不断训练学习修正各个神经元的权重，拟合需求规律，从而指导解决需求问题。
