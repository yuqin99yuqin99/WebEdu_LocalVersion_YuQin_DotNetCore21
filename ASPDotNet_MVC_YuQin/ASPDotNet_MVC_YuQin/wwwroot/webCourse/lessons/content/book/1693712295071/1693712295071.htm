注：示例代码是“客户端浏览器”互动“服务端AIGC已经训练学习的智能数据”的实现。SS的向量数据库功能、.Net平台的ML.Net扩展平台封装各种算法则是必然趋势，将在后续持续跟进相关技术的继承利用、创新研发。

 

namespace SemanticKernel_AI_LLM_Try.Controllers

{

    public class AskGPTTranslationController : Controller

    {

        [HttpGet(Name = "AskGPTTranslation")]

        public async Task<string> AskGPTTranslation(String inputString)

        {

//inputString在此传入参数是System.String字符串对象类型的对象实例"教育数据框架驱动的教师机器人辅助的教育决策支持系统"。

IKernel iKernel = Kernel.Builder.Build();//创建一个IKernel对象类型的iKernel对象实例

            ISKFunction llm = iKernel.CreateSemanticFunction("请扮演教育决策支持系统专家，告诉我"+inputString+"的涵义。"); //创建一个SemanticFunction实现模型的调用。SemanticFunction函数的输入是AIGC宿主的Prompt语言（目前Prompt语言最好是遵守RICIO语法框架）。在此的Prompt语言调用AIGC宿主返回回答Answer。当然也可进一步调用其他各种功能。

            SKContext? responseFromLLM = await llm.InvokeAsync(inputString);

            return responseFromLLM.Result;

        }

    }

}

 

神经元多层网络的已经训练学习的概率性推断统计AIGC的定量思维，涉及6.2.5节所述及其扩展的系统的定量知识体系，可以参考图 6‑29所示相关的ChatGPT/Qwen等等开源代码[[3]]、[[4]]、[[5]]。其中：

·       图 6‑29(a)所示的神经元多层网络是基础。三层神经网络可以模拟任何智能建构生成，不过层次深度越多，正确性一般越高，当然，参数也就越多，计算消耗也就越大（例如ChatGPT就有1750亿多个参数）[[6]]。

·       图 6‑29(b)所示的Transformer是核心。Transformer自身就可看作是一个独立的面向AIGC领域的MVC架构/MVC宿主[[7]]、[[8]]、[[9]]。本书的教育软件案例MVC可以调用现有的Transformer，快捷实现AIGC教育应用。
